# **App Name**: Rigel

## Core Features:

- Vector Bone Manipulation: Interactive skeletal system with draggable joints for precise 5-point character rigging and pose creation.
- AI Motion Interpolator Tool: An AI-powered feature that uses a predictive model to synthesize intermediate 'tween' frames between keyframes for fluid character motion.
- Sequence Timeline Manager: Multi-frame recording interface allowing users to organize, delete, and re-order character states in a professional animation timeline.
- Dynamic FPS Controller: Variable-rate playback engine providing real-time previews of recorded sequences from 1 to 60 FPS.
- Sprite Sheet Compiler: Automated asset exporter that compiles individual animation frames into a high-density, game-ready PNG sprite sheet.
- Pose Template Library: Curated library of biomechanical action presets such as locomotion cycles, defensive stances, and jump curves to jumpstart animation.

## Style Guidelines:

- Primary Color: Kinetic Cobalt (#5E81F3) - a vibrant, high-energy blue that evokes digital precision and professional creativity.
- Background Color: Obsidian Navy (#14171F) - a deep, desaturated dark tone that provides high contrast for neon joint points and grid systems.
- Accent Color: Glacier Azure (#94D4F1) - a cool, airy analogous shade used for highlighting active states and predictive paths.
- Main font: 'Space Grotesk' - a proportional sans-serif with a machined, technical aesthetic perfectly suited for the coordinate displays and lab-like interface of a rig editor.
- Lean, monolinear geometric icons inspired by drafting software, featuring sharp corners and a consistent 2px stroke weight.
- A sidebar-focused workstation layout featuring a primary centralized canvas with retractable control drawers for precision numeric input and timeline scrubbers.
- Snappy, physics-based transitions for menu expansions and frame preview cards, reflecting the motion-oriented nature of the app.
